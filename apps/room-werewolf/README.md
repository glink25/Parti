# 狼人杀 · 月夜审判

这是一个 6–12 人狼人杀房间，产品结构沿用“谁是卧底”的双入口设计，但拥有独立的角色、昼夜状态机和月夜哥特视觉。

## 模式

- **联机模式**：Worker 作为上帝，私密发牌并依次推动丘比特、守卫、狼人、女巫、预言家、警长竞选、白天放逐和胜负结算。
- **牌盒模式**：一台设备轮流验牌；真人主持实际流程，页面只记录死亡、情侣和胜负。

联机夜间每个子阶段要求所有存活玩家使用相同界面提交，只有对应角色的操作会被采纳。公开快照不包含身份、情侣、技能选择、药剂或验人结果。

## 房规与阵容

房主可在 6–12 人推荐阵容上调整身份牌，并配置情侣阵营、女巫自救与双药、守卫连守、同守同救和猎人毒死开枪规则。开局必须满足总牌数一致、至少一狼一民一神、神职不重复。

死亡默认不翻牌。好人屠狼获胜，狼人屠边获胜，第三方情侣需共同存活并消灭其他玩家。

## 状态与恢复

公开状态只保存当前阶段、提交人数、死亡、警长、公开票型和结算。秘密保存在 Worker 模块内存；房主运行环境从快照恢复后会回到等待状态并要求重新发牌。本版本不设计跨版本数据迁移。
