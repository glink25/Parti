# Endwell art assets

Runtime-safe copies selected from the ignored `dev-only` art packs. The game never references `dev-only` directly.

- `characters/`: player and ally source sprites
- `monsters/`: source silhouettes for the three ruins enemies
- `world/`: packed Tiny Dungeon sheet used for world and sprite rendering
- `effects/`: bounded smoke, flash, and explosion frame sequences
- `ui/`: inventory and HUD source icons

The two `KENNEY_*_LICENSE.txt` files are the original CC0 license notices.
