# Magicka 法术特殊效果手册

本文档整理 `room-magicka` 中所有特殊配方、卷轴配方、元素组合强化、状态反应与环境效果的玩法说明，供设计与测试参考。

---

## 基础规则

### 元素与队列

- 共七种元素：**岩、火、冰、生命、雷、水、盾**。
- 施法时按顺序选择最多 **4 个**元素，允许重复。
- **元素顺序决定法术结果**：完全匹配特殊配方时释放对应法术；否则以第一个元素为基础，后续槽位叠加组合强化。

### 法术载体

| 载体 | 说明 |
| --- | --- |
| 投射物 | 飞向瞄准方向，碰撞后结算，可穿透多个目标 |
| 扇形 | 以角色朝向为轴的扇区，一次命中范围内所有目标 |
| 射线 | 需持续按住；吟唱完成后喷射，按间隔持续伤害 |
| 定点 | 吟唱结束后在瞄准点进行范围结算 |
| 区域 | 在瞄准点结算，并可生成持续环境区域 |
| 全场 | 作用于当前房间内所有目标，常伴随全场环境 |
| 自身 | 直接作用于施法者，无需瞄准落点 |

### 解析优先级

1. 背包持有对应卷轴，且元素组合完全匹配，且卷轴冷却已结束 → **卷轴法术**
2. 元素组合完全匹配特殊配方表 → **特殊配方**
3. 否则 → **混合法术**（首元素基础 + 2/3/4 槽强化）

### 发现与卷轴

- 特殊配方首次成功释放后写入法术书；开局默认已知 **复活术**。
- 卷轴携带即可在法术书中预览对应组合，无需先发现。
- 卷轴冷却 **20 秒**，按卷轴类型共享；冷却中输入相同组合将解析为混合法术。
- 卷轴掉落来源：Boss 必掉、精英较高概率、商人/宝箱/诅咒宝箱低概率。

## 特殊配方（29 种）

以下组合须**严格按顺序**输入。表中数值为设计基准，实际受装备加成影响。

### 辅助、位移与防护

#### 复活术
- **组合**：雷 → 生命 → 雷
- **载体**：定点
- **效果**：范围内倒地友方恢复至 **25% 最大生命**并起身
- **备注**：开局默认已知；普通治疗无法救起倒地队友

#### 瞬间移动
- **组合**：雷 → 盾 → 雷
- **载体**：定点
- **效果**：向瞄准方向瞬移，最远 **600**；路径须可走，前方无空间则失败

#### 消除术
- **组合**：生命 → 冰 → 盾
- **载体**：区域
- **效果**：清除范围内所有状态（燃烧、潮湿、寒冷等）及减速、眩晕等控制

#### 加速
- **组合**：雷 → 盾 → 火
- **载体**：自身
- **效果**：移速提升 **70%**，持续 **6 秒**

#### 生命之泉
- **组合**：水 → 生命
- **载体**：区域
- **效果**：治疗 **35**，附加潮湿

#### 连锁治疗
- **组合**：生命 → 雷 → 生命
- **载体**：定点
- **效果**：范围内治疗 **32**
- **备注**：名称含「连锁」，当前版本为普通范围治疗，尚无额外弹跳逻辑

#### 岩石护盾
- **组合**：盾 → 岩
- **载体**：自身
- **效果**：护盾 **2 层**，持续 **9 秒**

#### 火焰护盾
- **组合**：盾 → 火
- **载体**：自身
- **效果**：护盾 1 层 + **防火**（免疫燃烧），各持续 **9 秒**

#### 寒冰护盾
- **组合**：盾 → 冰
- **载体**：自身
- **效果**：护盾 1 层 + **防冻**（免疫寒冷与冻结），各持续 **9 秒**

#### 接地护盾
- **组合**：盾 → 雷
- **载体**：自身
- **效果**：护盾 1 层，持续 **9 秒**；潮湿状态下队列含盾时可安全加入雷元素

#### 流水护盾
- **组合**：盾 → 水
- **载体**：自身
- **效果**：护盾 1 层 + **防潮**（免疫潮湿），各持续 **9 秒**

#### 烈焰喷射

### 环境与领域

#### 雷击术
- **组合**：雷 → 火 → 火 → 雷
- **载体**：定点
- **效果**：在落点生成 **3 秒**雷击领域，半径约 **145**；领域内每 **0.5 秒**造成 **24** 伤害

#### 暴风雪
- **组合**：水 → 冰 → 冰 → 水
- **载体**：全场
- **效果**：当前房间 **10 秒**暴风雪，覆盖整个房间；周期性施加寒冷与减速

#### 全场降雨
- **组合**：火 → 水 → 水 → 火
- **载体**：全场
- **效果**：当前房间 **12 秒**降雨；全场周期性使玩家与怪物 **潮湿**

#### 蒸汽云
- **组合**：火 → 水
- **载体**：区域
- **效果**：即时火伤 **18**；落点生成 **5.5 秒**蒸汽区，区内每跳 **3** 伤害

#### 冰墙
- **组合**：冰 → 盾 → 盾 → 冰
- **载体**：区域
- **效果**：落点 **6.5 秒**冰墙，半径约 **110**；阻挡玩家与怪物通行，区内寒冷 + 减速

#### 雷暴领域
- **组合**：雷 → 水 → 雷 → 盾
- **载体**：区域
- **效果**：**6 秒**雷暴区，半径约 **155**；每跳对潮湿目标 **15** 伤，否则 **10** 伤

#### 重力井
- **组合**：岩 → 盾 → 雷 → 岩
- **载体**：区域
- **效果**：**4.2 秒**引力井，半径约 **170**；持续将目标拉向中心，**结束时**对范围内造成 **35** 爆发伤害

#### 火焰环
- **组合**：火 → 盾 → 火
- **载体**：自身
- **效果**：以施法者为中心 **5 秒**火环，半径约 **135**；每跳 **8** 火伤并点燃

#### 生命屏障
- **组合**：生命 → 盾 → 生命 → 盾
- **载体**：区域
- **效果**：**6.5 秒**治疗区，半径约 **145**；玩家每跳 **+6** 生命并获短时护盾（怪物也会回血）

## 卷轴配方（4 种）

须背包持有对应卷轴。吟唱约 **650** 毫秒，冷却 **20 秒**。

| 卷轴 | 元素组合 | 载体 | 效果 |
| --- | --- | --- | --- |
| 超新星爆发 | 雷 → 生命 → 盾 → 火 | 全场 | 当前房间所有存活实体损失 **50% 当前生命** |
| 均衡术 | 岩 → 盾 → 冰 → 生命 | 全场 | 当前房间所有存活实体生命设为 **50% 最大生命** |
| 湮灭术 | 岩 → 冰 → 生命 | 定点 | 范围内**恰好一个**目标损失 **99% 当前生命**（Boss 为 **50%**）；目标数不对则施法失败 |
| 黑洞 | 盾 → 岩 → 岩 → 盾 | 定点 | 落点生成 **8 秒**黑洞，半径约 **340**；外圈强力牵引，内圈每 **0.5 秒** **12** 伤害，无差别作用于所有实体 |

**卷轴共性**：造成伤害会打破隐身；卷轴伤害不受感电增伤等特殊加成。

## 状态效果

### 状态一览

| 状态 | 说明 |
| --- | --- |
| 潮湿 | 为元素反应做准备；降雨、水洼、多种水系法术可施加 |
| 燃烧 | 每秒受到层数 ×2 的持续伤害 |
| 寒冷 | 减速；累计 3 层且不在冻结免疫期内 → **冻结** |
| 冻结 | 强控约 1.5 秒，随后进入约 3.5 秒冻结免疫 |
| 感电 | 下次受到伤害时额外 +15% 并清除感电 |
| 护盾 | 吸收一次伤害（按层数消耗） |
| 加速 | 移速提升 |
| 防火 / 防冻 / 防潮 | 对应负面状态无法施加 |

### 元素反应（伤害加成）

| 目标已有状态 | 新伤害元素 | 结果 |
| --- | --- | --- |
| 潮湿 | 火 | 伤害 ×1.5，显示「蒸发」，清除潮湿 |
| 潮湿 | 雷 | 伤害 ×1.45，显示「导电」，清除潮湿 |
| 冻结 | 岩 | 伤害 ×1.6，显示「碎裂」，清除冻结 |

### 状态互斥

- 施加潮湿会熄灭燃烧。
- 目标潮湿时，燃烧无法施加。
- 潮湿玩家直接选雷（且队列无盾）会自身感电、眩晕并打断施法。

| 类型 | 说明 |

## 其他战斗规则

- **友军**：不受友方直接伤害，可接受治疗、状态与控制。
- **亡灵标签**：治疗对亡灵敌人视为伤害。
- **Boss 机制护盾**：遗迹守卫、腐沼之心、熔炉泰坦在机制未完成时仅受约 **20%** 法术伤害；玩家站在机制点位可推进破解。
- **射线**：默认被怪物阻挡；满功率维持约 1.5 秒后逐渐衰减，最低约 55%。
- **装备**：可缩短吟唱、扩大范围、提升元素/配方威力；戒指可令射线反射或穿透。
