# 元素训练场

1–4 人合作的 Magicka 玩法框架 Demo。玩家在固定战斗带中点击移动，依次选择最多四个元素，再按住场景目标进行吟唱或引导。首版使用基础碰撞盒，怪物会持续生成。

## 同步模型

- 每个客户端结算自己的移动、施法碰撞、受伤、状态和倒地。
- 房主客户端生成和推进怪物及环境，并通过 Worker 发布共享结果。
- 所有玩家上报自己算出的怪物伤害；房主按事件去重后更新共享怪物生命。
- Worker 仅保存快照、转发事件和处理房间阶段，不验证距离、轨迹或伤害合法性。

## 操作

- 空元素队列：点击场地直线移动。
- 非射线法术：按住场景瞄准，松手提交；法术在松手后的吟唱时间结束时自动触发。吟唱期间可继续准备元素，但不能释放下一法术。
- 射线法术：按住后先吟唱，随后持续喷射并跟随手指转向；松手立即结束。
- 触屏点击底部七元素按钮；桌面端可用数字键 `1–7`。
- `Backspace` 撤销最后一个元素，`Escape` 清空队列。
- 潮湿时直接加入雷会使自己感电；先选择盾可组成接地法术。

怪物分为近战、远程和水系混合三类，具有可见前摇和独立攻击冷却。怪物硬上限随玩家数依次为 8、12、16、20。岩、水、冰和雷系法术分别可产生击退/击飞、推力、减速/冻结和眩晕。
